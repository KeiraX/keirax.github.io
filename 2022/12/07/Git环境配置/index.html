重置Git账号密码

  • 来源：git提交代码权限问题报错记录
  • 报错信息
  • 问题分析
    • 用户没有赋权
    • 用户密码更改
    • Tfs账号问题
  • 解决方案
    • 在管理员模式的GitBash.exe中执行命令
      git config --system --unset credential.helper
    • 永久保存密码
      git config --global credential.helper store

  • 来源：TortoiseGit save user authentication / credentials - Stack Overflow
